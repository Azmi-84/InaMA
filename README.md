# InaMA

InaMA is my personal project to create my academic assignments and other stuff using LaTeX. LaTeX is my favourite tool for writing as eliminate many hassles that have to be faced in Microsoft Word. I started using it by the inspiration of one of our faculty members named [Dr. Muhammad Rizwanur Rahman, D.I.C.](https://mpe.iutoic-dhaka.edu/profile/Muhammad-Rizwanur/education), who was using it to make his presentation slides. When he first gave us the first assignment of our Fluid Mechanics-II course, I started using it. Now, in my research work, this is the go-to tool for me.
